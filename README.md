# IIT Chatbot 🎓

AI-powered chatbot for IIT university website.

## Tech Stack
- FastAPI (backend)
- React + Mantine (frontend)
- PocketBase (database)

## Team
- Yessin Jribi

## To Start
 - terminal 1
 cd backend
 venv\Scripts\activate
 uvicorn main:app --reload

 - terminal 2
 cd frontend
 npm run dev
